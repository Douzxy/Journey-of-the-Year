# Photos Directory

Place your photos here organized by month:

```
photos/
├── january/
│   ├── 1.jpg
│   ├── 2.jpg
│   └── 3.jpg
├── february/
│   ├── 1.jpg
│   ├── 2.jpg
│   ├── 3.jpg
│   └── 4.jpg
├── march/
│   └── ...
└── december/
    └── ...
```

## How to Add Photos

1. Create a folder for each month (all lowercase)
2. Name your photos with numbers: `1.jpg`, `2.jpg`, `3.jpg`, etc.
3. Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`
4. Recommended size: ~800x600px for optimal loading

The website will automatically display placeholder cards if photos are not found.
