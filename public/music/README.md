# Music Folder

Place your background music file here:

**Required filename:** `background.mp3`

## Recommended Music

- Instrumental, chill, or lo-fi music
- Duration: 3-5 minutes (will loop automatically)
- Format: MP3
- Volume will be set to 50% by default

## Example Sources (Royalty Free)

- https://pixabay.com/music/
- https://www.bensound.com/
- https://freemusicarchive.org/

## Note

The music player will:

1. Ask for permission on first visit
2. Remember user's preference in localStorage
3. Allow play/pause anytime via control button
